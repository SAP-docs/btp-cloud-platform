# Cannot Find Kyma Resource in Kyma Dashboard



## Symptom

When I access Kyma dashboard, I cannot find some Kyma resources in the left panel.



## Cause

The relevant Kyma module is not added. For example, if you're missing the Function resource, you must add the Serverless module first. In the case of Subscription, you must add the Eventing module. For the list of all available modules, see [Kyma Modules](kyma-modules-0dda141.md).



## Solution

See [Adding and Deleting a Kyma Module](../50-administration-and-ops/adding-and-deleting-a-kyma-module-1b548e9.md#loio1b548e9ad4744b978b8b595288b0cb5c).
