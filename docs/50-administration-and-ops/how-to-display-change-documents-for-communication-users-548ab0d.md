# How to Display Change Documents for Communication Users



<a name="loio548ab0db801b4207b6e4854d93b8d094__section_ContextChangeDocsCU"/>

## Context

To get an overview of the changes that were made to communication users, you can display a list of change documents by completing the following steps:



<a name="loio548ab0db801b4207b6e4854d93b8d094__section_StepsChangeDocsCU"/>

## Procedure

1.  Click *Display Changes* in the *Maintain Communication Users* list view if you want to display change documents for several communication users. Alternatively, click *Display Changes* in the *Maintain Communication User* detail screen if you want to display the change documents for one communication user.
2.  Select the required change categories. The following change categories are available:
    -   *User Created*
    -   *User Deleted*
    -   *Password Status*
    -   *Password*
    -   *Lock Changed*
    -   *User Alias Changed*
    -   *Communication Scenario Assignment Changed*
    -   *Certificate*

3.  If required, filter for certain communication users in the *User ID* area.
4.  If required, filter for certain date ranges or for specific points in time in the *Changed At* area.
5.  If required, filter for changes made by specific users in the *Changed By* area.
6.  Click *Go* to view the relevant change documents.
