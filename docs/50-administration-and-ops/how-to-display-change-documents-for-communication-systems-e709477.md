# How to Display Change Documents for Communication Systems



<a name="loioe70947725abf4792bac999eaf047739e__section_ContextChangeDocsCS"/>

## Context

To get an overview of the changes that were made to communication systems, you can display a list of change documents by completing the following steps:



<a name="loioe70947725abf4792bac999eaf047739e__section_StepsChangeDocsCU"/>

## Procedure

1.  Click *Display Changes* in the *Communication Systems* list view if you want to display change documents for several communication systems. Alternatively, click *Display Changes* in the *Communication System* detail screen if you want to display the change documents for one communication system.
2.  Select the required change categories. The following change categories are available:
    -   *Communication System*
    -   *Users for Inbound Communication*
    -   *Users for Outbound Communication*
    -   *Business Partners* 

3.  If required, filter for certain communication systems in the *System ID* area.
4.  If required, filter for certain date ranges or for specific points in time in the *Changed At* area.
5.  If required, filter for changes made by specific users in the *Changed By* area.
6.  Click *Go* to view the relevant change documents.
