# How to Import and Export Business User Groups



<a name="loioc3d16efe55894fe1ae76b2e6fe90ac15__HowToImportExportBusinessUserGroups_context"/>

## Context

You can use the *Export Software Collection* app and the *Import Collection* app to transport business user groups from your source system to your target system.

We recommend that you do not change the business user group after it has been transported. It should not be changed locally but only updated with a new transport. Local changes should only be carried out in urgent cases.



<a name="loioc3d16efe55894fe1ae76b2e6fe90ac15__HowToImportExportBusinessUserGroups_steps"/>

## Procedure

1.  In your source system, open the*Export Software Collection* app.

2.  Create a software collection and give it a unique name.

3.  Click *Add Items* and select the business user groups you want to export.

4.  Click *Export*to export the required business user groups to your target system. You can view in the log if the export has been finished.

5.  In your target system, open the *Import Collection* app.

6.  Select the previously exported software collection and click *Import* to import the required business user groups to your target system. You can view in the log if the import has been finished.
