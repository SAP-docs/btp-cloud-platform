# How to Display Change Documents for Communication Arrangements



<a name="loio4a62636e360a4d04a7422bcd0ad4cf22__section_ContextChangeDocsCA"/>

## Context

To get an overview of the changes that were made to communication arrangements, you can display a list of change documents by completing the following steps:



<a name="loio4a62636e360a4d04a7422bcd0ad4cf22__section_StepsChangeDocsCA"/>

## Steps

1.  Click *Display Changes* in the *Communication Arrangements* list view if you want to display change documents for several communication arrangements. Alternatively, click *Display Changes* in the *Communication Arrangment* detail screen if you want to display the change documents for one communication arrangement.
2.  Select the required change categories. The following change categories are available:
    -   *Communication Arrangement*
    -   *Communication Arrangement Root Property*
    -   *Communication Arrangement Root Property Value*
    -   *Communication Arrangement Root Property Structure Instance*
    -   *Communication Arrangement Inbound Service Defaults*
    -   *Communication Arrangement Inbound Service*
    -   *Communication Arrangement Inbound Service Property*
    -   *Communication Arrangement Inbound Service Property Value*
    -   *Communication Arrangement Outbound Service Defaults*
    -   *Communication Arrangement Outbound Service*
    -   *Communication Arrangement Outbound Service Property*
    -   *Communication Arrangement Outbound Property Value*

3.  If required, filter for certain communication arrangements in the *Arrangement Name* area.
4.  If required, filter for certain date ranges or for specific points in time in the *Changed At* area.
5.  If required, filter for changes made by specific users in the *Changed By* area.
6.  Click *Go* to view the relevant change documents.
