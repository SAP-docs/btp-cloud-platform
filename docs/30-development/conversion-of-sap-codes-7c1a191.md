# Conversion of SAP Codes

Learn how to use the methods of class `CL_CONVERSION_EXT_INT` to convert SAP codes to external codes and vice versa.



## Context

Communication in heterogeneous IT landscapes uses external data formats \(ISO\). SAP systems use a different internal format for data such as currencies, languages, and units of measurement. Therefore, data must be converted into the appropriate format during communication.
